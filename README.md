# E-Commerce Sales Analysis Dashboard

This project is an interactive Excel dashboard created to analyze e-commerce sales performance.  
It provides insights into sales, profit, order quantity, and category performance using visual charts and KPIs.

## Dashboard Overview
The dashboard includes the following key metrics:

- Total Sales
- Total Profit
- Quantity Sold
- Number of Orders
- Profit Margin
- Year-over-Year Growth (YOY)

## Features
- Monthly Sales and Profit Analysis
- Category-wise Profit Breakdown
- Category-wise Sales Share
- State-wise Sales Distribution Map
- Top 5 Sub-Categories by Sales
- Interactive Filters for:
  - Year
  - Region
  - Customer Segment

## Visualizations Used
- Combo Chart (Sales vs Profit)
- Waterfall Chart (Category Profit Analysis)
- Donut Chart (Category Sales Share)
- Map Chart (Sales by State)
- Bar Chart (Top 5 Sub Categories)

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Visualization

## File Included
- `Ecommerce Sales Analysis.xlsx` – Main Excel dashboard file

## Purpose of the Project
This project demonstrates how Excel can be used for business intelligence and data visualization to analyze sales data and identify performance trends.
